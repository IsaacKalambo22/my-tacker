# Tech Learning Progress Tracker

A personalized web application for tracking your progress in learning new technologies and skills.

## Features

- **Track Multiple Subjects**: Keep track of multiple technologies or skills you're learning
- **Structured Learning Path**: Each subject follows a structured learning path with 5 phases:
  - Foundation & Planning
  - Core Learning
  - Hands-On Application
  - Community & Professional Development
  - Assessment & Certification
- **Progress Tracking**: Visual progress bars and statistics
- **Data Export**: Export your learning data as JSON
- **Responsive Design**: Works on desktop and mobile devices
- **Local Storage**: All data is stored locally in your browser

## How to Use

1. Open `index.html` in your web browser
2. Click "Add New Subject" to start tracking a new technology or skill
3. Fill in the details:
   - Subject/Technology Name
   - Priority Level
   - Target Completion Date
4. Check off tasks as you complete them
5. Track your overall progress in the statistics panel

## File Structure

- `index.html`: Main HTML structure
- `css/styles.css`: Styling for the application
- `js/script.js`: JavaScript functionality

## Customization

You can customize the learning template in the `script.js` file by modifying the `learningTemplate` object.

## Data Privacy

All data is stored locally in your browser using localStorage. No data is sent to any server.
